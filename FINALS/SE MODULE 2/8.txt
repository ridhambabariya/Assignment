8. In how many ways can a CSS be integrated as a web page?

Ans. ---> CSS can be integrated as a web page in 3 main ways 
1.External CSS
 e.x. --->
<head>
    <link rel="stylesheet" type="text/css" href="styles.css">
</head>

2.Internal CSS
 e.x. --->
 <head>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f2f2f2;
        }
    </style>
</head>

3.Inline CSS
 e.x. --->
 <p style="color: red; font-size: 16px;">This is a paragraph with inline CSS.</p>
